# ME Storage Reshuffle
Anyone far enough into Applied Energistics will know how potentially painful it can be to defragment your digital storage, so 2.9 comes with a new solution! Introducing the <Color id="GREEN">Storage Reshuffler</Color> <ItemImage id="appliedenergistics2:tile.BlockStorageReshuffle"/>!

The basic functionality of this block is fairly simple, having 3 main modes:

## Reshuffle Mode
This mode is the "main" attraction of this block, extracting items and re-injecting them into existing cells, taking into account priorities to help optimize cell usage and reduce fragmentation. When running, it will show the amount extracted and injected, as well as time spent and types/items moved.

## Scan Mode
This mode is super simple, but can be vitally useful. It simply scans the network for duplicate partitions and displays how many it finds, as well as the drives containing them.

## Cell Health Mode
The cell health mode works as a basic diagnostic screen for your digital storage, showing the usage of all cells in the network, sortable by either fill percentage or storage size in bytes.
