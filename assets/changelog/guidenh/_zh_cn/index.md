# 我现在看到的是什么？
你现在读到的这份更新日志，是通过 2.9 新加入的模组 [GuideNH](./misc/guide.md) 制作的！这个模组未来会逐步加入大量关于整包不同内容的指南，并提供很多展示思路与机制的方式。想了解更多的话，可以点进对应页面看看。

如果你也想参与贡献，可以前往 Discord，找到 `#guide-dev` 频道加入。
