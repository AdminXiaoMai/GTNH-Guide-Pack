# 资源包更新提示

我们知道你们肯定会装资源包，所以现在游戏里也有方便的更新提醒了！以后不用再碰到“某个地方其实早就修好了，但你这边贴图还是坏的”这种事。

<Color id="GREEN">资源包更新检查器</Color> 会读取你当前启用的资源包，并去检查 GitHub Releases 上是否有更新且兼容的新版本。

当然，这需要资源包作者主动接入；但只要检测到更新，游戏里就会给你发一条带可点击链接的聊天消息，直接带你跳到最新兼容版本的发布页。
