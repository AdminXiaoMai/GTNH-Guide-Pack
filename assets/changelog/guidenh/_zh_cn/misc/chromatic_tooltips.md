# 彩色提示框

<Color id="GREEN">彩色提示框</Color> 模组现已加入，它会彻底重做提示框的外观！

现在提示框可以拥有完全不同的表现形式，比如角落里的物品图标、渐变边框、出现动画、更清晰的信息分隔，等等。

而且这些样式还能通过资源包进行调整。你可以自己看看，哪些提示框已经换了新样子。
