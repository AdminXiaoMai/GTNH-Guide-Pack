# 任务完成度追踪

任务书现在会显示你总共完成了多少个任务，以及整体完成百分比！每个章节的完成度文本现在也会显示类似的百分比。

需要说明的是，它不会统计某些“隐藏”任务，或者一般来说任何当前不能直接看到的任务。
