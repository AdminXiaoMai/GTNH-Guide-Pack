# 主菜单背景

如果你已经加载进过 2.9，可能早就注意到了：这次主菜单新增了 23 张全新的背景图，而且素材直接来自 NASA/ESA！

其中包括哈勃空间望远镜、詹姆斯 韦伯空间望远镜，以及最新的 Artemis II 载人绕月飞行任务拍摄的照片。

借着这次改动，我们把原先那些插画风 / 素材风的背景图移除了，改用真实的太空摄影图像。当然，那张作为整包标志、已经陪了 GTNH 十年的经典主菜单背景仍然保留。

另外，主菜单本身也顺手做了一点视觉重整，让信息更清晰。
