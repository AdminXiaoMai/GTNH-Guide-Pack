# 多方块诊断反馈

从 2.9 开始，所有 GregTech 多方块都会准确告诉你：到底缺了哪些仓室、总线、机械方块等等，而且还会告诉你缺在什么位置。

如果所有机械方块都已经摆好了，它就会继续显示缺失的仓室；如果某个位置缺的是机械方块，它还会告诉你那个位置可以放什么，并附带一个按钮，直接在世界里高亮那一格！
