# 现代化 Tab 列表

现代风格的 “tab 列表” 现在也被回移植进来了，让联机时的体验更利落！

现在只要在服务器上调整 `serverutilities.cfg`，你就可以配置：

- 玩家头像
- 延迟数值
- 动态列数
- 页眉和页脚
