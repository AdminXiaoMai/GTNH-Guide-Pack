# 披风切换

如果你曾经给整包捐过款，或者参与过开发，那么…… <Color id="GREEN">__非常感谢！__</Color>

现在可以直接在游戏内使用 `/cape` 命令切换披风了！你的选择会保存在客户端里，所以不用在每个不同存档里反复重设。

如果你觉得自己理应有披风但实际上没有，欢迎去 [Discord](https://discord.gg/gtnh) 联系开发组。
