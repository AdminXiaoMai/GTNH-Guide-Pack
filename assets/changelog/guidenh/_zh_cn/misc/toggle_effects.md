# 效果显示切换

这是个小改动，但确实挺舒服。

现在你可以在 `Hodgepodge options -> tweaks` 中，通过 `showInventoryEffectIcons` 开关切换物品栏里的效果图标。如果你觉得那排显示药水效果的横幅太挤眼，现在终于可以直接把它关掉了。
