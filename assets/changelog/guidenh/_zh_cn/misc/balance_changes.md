# 平衡性改动

和任何一次大型更新一样，2.9 也在整包各处投入了不少精力，目标是让玩法更有趣、路线选择更多。这通常意味着会做大量平衡调整，让原本偏弱的选择更值得用，也让那些“强得太出风头”的选项稍微收敛一点。

> [!NOTE]
> 完整的平衡改动列表既太长、也太容易带主观判断，不适合全塞在这里。所以这页不会包含诸如多方块造价变动（至少不含结构本体）这类细项，也不会把那些通过新增路线直接带来平衡变化的内容全算进来（例如 [CropsNH](../crops/crops.md)）。由于改动数量实在太多，想知道 __全部__ 变化，唯一现实的办法还是去逐条翻 GitHub changelog。因此下面这份列表只是从 2.9 中挑出来的一部分较大改动。

- 巨型蒸馏塔（HV -> LuV）、巨型石油裂化塔（HV -> LuV）、巨型工业高炉（现为放热炉床）（HV -> ZPM）、巨型真空冷冻机（现为吸热冰箱）（HV -> ZPM）
- Hypogen（UIV -> UEV），鸿蒙之眼处理现在需要 UXV 电压
- 基于钛的四氯化钛配方（10 秒 HV -> 60 秒 EV）
- 一系列 [结构重做](../reworks/reworks.md) 同时带来了各种平衡调整
- 寒霜之液现在使用青金石粉，而不是红石
- Railcraft 给水侧槽新增了配方
- MV 力场发生器现在改用钛，而不是钨钢
- 基础 SMD 配方现在需要 HV 组装机，而不再是 MV
