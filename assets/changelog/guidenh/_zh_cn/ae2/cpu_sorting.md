# CPU 排序

如果你一直希望自己的合成 CPU 不要只能按名字排序，那现在这个愿望实现了。

CPU 现在可以按以下多种条件排序：

- CPU 名称
- 存储容量
- 协处理器数量
- 合成状态
- 自动化来源（玩家、混合、非玩家）

以上每一项都支持按升序或降序排序。
