# Forbidden Magic 法杖部件强化

下面这些 <Color id="BLUE">Forbidden Magic</Color> 的法杖部件在 2.9 中得到了 <Color id="GREEN">强化</Color>：

- <ItemImage id="ForbiddenMagic:WandCaps:3"/>：<Color color="#FFAA00AA">vis 折扣</Color> 从 <Color color="#FFFF55FF">10%</Color> 提升到 <Color color="#FFFF55FF">15%</Color>，<Color color="#FFAA00AA">合成成本</Color> <Color color="#FFFF55FF">降低</Color>，并且 <Color color="#FFAA00AA">下调到</Color> <Color color="#FFFF55FF">MV</Color>
- <ItemImage id="ForbiddenMagic:WandCaps:5"/>：<Color color="#FFAA00AA">vis 折扣</Color> 从 <Color color="#FFFF55FF">20%</Color> 提升到 <Color color="#FFFF55FF">30%</Color>
- <ItemImage id="ForbiddenMagic:WandCores:7"/>：<Color color="#FFAA00AA">合成成本</Color> <Color color="#FFFF55FF">降低</Color>
- <ItemImage id="ForbiddenMagic:WandCores:11"/>：<Color color="#FFAA00AA">vis 容量</Color> 从 <Color color="#FFFF55FF">100</Color> 提升到 <Color color="#FFFF55FF">200</Color>
- <ItemImage id="ForbiddenMagic:WandCores:13"/>：<Color color="#FFAA00AA">vis 容量</Color> 从 <Color color="#FFFF55FF">250</Color> 提升到 <Color color="#FFFF55FF">500</Color>

为了平衡这些强化，它们制作成法杖时所需的 vis 成本也相应 <Color id="RED">提高</Color> 了。这样一来，它们在各自可用阶段里会更有竞争力，也更像是一份在投入 <Color id="BLUE">植物魔法</Color> 之后得到的合理回报。

它们把 <Color id="BLUE">魔力</Color> 转换为 <Color color="#FFAA00AA">vis</Color> 的效率和速度则保持不变。
