# 特斯拉电塔界面

<Color id="GREEN">特斯拉电塔</Color> 现在有了一个非常好用的新界面，而且功能相当多：

- 新的参数管理面板
- 实时负载条，用来显示当前吞吐量
- 可配置的负载变化图表
- 热力图，用来定位你最大的特斯拉耗电端

这个界面细节实在太多，没法在这里一一展开，直接自己进去摸一遍会更快。
