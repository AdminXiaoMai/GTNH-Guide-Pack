# 个人维度外观设置

是不是已经看腻了草坪基地，或者脚下悬着一整片虚空？现在 <Color id="GREEN">个人维度传送门</Color> 也换上了全新的设计，甚至还内置了一个“城市街区”风格预设！

## 新功能：

- 全新的界面设计！
- 现在可以预览即将生成的地形
- 配置列表中的方块现在允许使用 metadata
- 创建维度时可用的功能进一步扩展
- 新增 __100+ 个装饰方块预设__，以及一套全新的地形预设

如果你整出了有趣、很酷，或者纯粹离谱的设计，欢迎发到 <a href="https://discord.gg/gtnh">Discord 服务器</a>！
