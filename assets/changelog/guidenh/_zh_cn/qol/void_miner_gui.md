# 虚空采矿场筛选

你有没有觉得，为一整排 <Color id="RED">虚空采矿场</Color> 往输入总线里塞指定矿石做筛选，整个过程非常折磨？

现在这些事情都能直接在虚空采矿场内部的一个统一界面里完成了。

打开新界面后，你可以按需对当前维度掉落池中的每一种矿石逐个开关筛选。这项改动还额外加入了：

- 不再需要那把烦人的老螺丝刀，就能在黑名单和白名单模式之间切换
- 使用 <Color id="GREEN">数据棒</Color> 复制与加载筛选配置
- 按名称搜索并高亮矿石
- 通过专用按钮一次性添加或移除 __全部__ 矿石

> [!NOTE]
> 这个新界面与旧搭法完全兼容；无论矿石是在输入总线里，还是在界面中被选中，都会参与筛选
