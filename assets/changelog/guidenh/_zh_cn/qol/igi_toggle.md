# InGame Info 切换

<Color id="GREEN">InGame Info</Color>（默认在左上角显示的信息）现在支持第二套配置文件了！

这意味着你可以保留两套不同的 HUD 布局，并且随时在它们之间切换。

只要额外添加一个可选文件（`\config\InGameInfo2.xml`），然后使用切换按键，就可以在 `主配置 -> 次配置 -> 关闭` 之间循环切换。
