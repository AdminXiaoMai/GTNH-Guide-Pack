# 混沌定位器

打到后期的人大多都知道，飞去混沌守卫所在位置这件事又远又折磨，更别提路上还会额外生成几百个区块。

混沌守卫总是生成在 <Color id="RED">混沌岛</Color> 上，而混沌岛的 x 和 z 坐标一定都是 10,000 的倍数（比如 x=10,000，z=0）。有了 <Color id="GREEN">混沌定位器</Color> <ItemImage id="gregtech:gt.Item_Chaos_Locator" />，这就不再是问题了。

只要使用这个物品，玩家就能在界面里输入两个数字，分别作为 x 和 z；随后它会把玩家传送到 `(10,000 x X 坐标, 10,000 x Z 坐标)` 的位置，从此不用再痛苦长途飞行了。

当然，到了那边以后等着你的那头怪物，就不是它能帮你解决的了。

而且这个功能还支持跨维度使用！
