# 覆盖板改动

这次对 GregTech 机器，以及它们的管道和线缆上使用的 <Color id="GREEN">覆盖板</Color> 做了几项改动，下面分别说明。

# 破坏机器时回收覆盖板

有时候，看起来最小的改动反而最有体感。

现在如果机器上装着覆盖板，只要在破坏时按住 Shift，覆盖板就会一起掉落。直接正常破坏则仍然保持原来的行为。

# 染色玻璃覆盖板

你有没有想过，把 **染色玻璃** 或其他半透明方块也贴到管道和机器上当覆盖板？

现在可以了。

只要把你喜欢颜色的染色玻璃和 AE2 线缆伪装板合成，然后把它贴到下面这些方块上：

- 管道
- 线缆
- 框架
- 机器
