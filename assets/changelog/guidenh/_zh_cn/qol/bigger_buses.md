# 更大的输入总线

更大尺寸的输入/输出总线这次主要有两项改动，下面分别说明。

# 更高等级的输入总线
<FloatingImage src="../assets/qol/uhv_bus.png" align="left" width="128" wrap="square" >
  <ImageAnnotation>
    UHV 输入总线的界面
  </ImageAnnotation>
</FloatingImage>

只要献祭掉超级总线，现在你终于可以一路合成到 UHV 级的 <Color id="GREEN">输入总线</Color> 和 <Color id="RED">输出总线</Color> 了。

和之前的总线一样，新总线依旧遵循 <Color color="#00ffff">n<sup>2</sup></Color> 个槽位的规则，其中 __n__ 是电压等级编号（ULV=1）。

已经合成或摆放出来的超级总线仍然会保留，但以后将无法再合成。

<br clear="all"/>

# 量子总线 / 压缩总线

作为非 AE2 物流系统整体翻修的一部分，游戏里还新增了一套新的输入/输出总线。<Color id="GREEN">量子总线</Color> <ItemImage id="gregtech:gt.blockmachines:2757" /> 和 <Color id="BLUE">压缩总线</Color> <ItemImage id="gregtech:gt.blockmachines:2751" /> 可以看作“普通” I/O 总线的强化版本，因为它们现在每个槽位都能容纳超过一组物品。
